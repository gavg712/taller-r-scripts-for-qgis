1.1 Contents

We are Gabo Gaona (Ecuador) and Antony Barja (Peru) and we have prepared this tutorial to show a little bit of the Processing R Provider add-on and how it can be useful for R and QGIS users. This time that we will share together we will try to build algorithms for the QGIS Processing toolbox. We hope that by the end of this tutorial you will have acquired enough skills to write your own tools for QGIS using the R language.

  • Duración: 2.5 horas
  • Idioma: Español
  • Modalidad: Virtual

Aim

Learn how to write and add new tools to the QGIS process box using the R programming language.
